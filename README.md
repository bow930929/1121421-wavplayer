# WAV 音效檔播放器

## 功能說明

| 按鈕 | 功能 |
|------|------|
| 瀏覽 | 開啟檔案對話方塊，選取 `.wav` 音效檔 |
| 播放一次 | 非同步播放所選音效檔一次 |
| 重複播放 | 循環播放所選音效檔，直到按下停止 |
| 停止播放 | 停止目前正在播放的音效 |
| 結束程式 | 關閉應用程式（會跳出確認對話框） |

## 注意事項

- 僅支援 `.wav` 格式的音效檔
- 切換播放模式前會自動停止當前音效
- 直接關閉視窗與點選「結束程式」皆會觸發關閉確認對話框
